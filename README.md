# cigario-com

Cigario is an ecommerce website used to sell cigars! It is built using Node and Express JS including other technologies!

The purpose of this project was to learn Javascript, Node JS, and Express JS.

With these technologies I've managed to: 
  - Allow the user, known as the customer, to register an account
  - Store user details in mySQL DB, whilst hashing user password using BCryptJS
  - Allow the user to log into its account
  - Incorporate sessions using passport JS
  - Allow the user to delete its account through user settings page 
  - Allow the user to change its account password through user settings page
  - Allow the user to logout out of its account
  - Incorporate Winston Logging to track user activity
  - Incorporate application testing using supertest
